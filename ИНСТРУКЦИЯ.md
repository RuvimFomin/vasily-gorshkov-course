# Инструкция по подключению к Google Таблице

## Шаг 1 — Создать Google Таблицу

1. Открыть [sheets.google.com](https://sheets.google.com)
2. Создать новую таблицу, назвать лист **Анкеты**
3. Добавить заголовки в первую строку (A1:I1):
   ```
   Дата | Email | Имя | Опыт | Телеграм | Запрос | Сложности | Готовность | Как давно знает
   ```
4. Скопировать ID таблицы из URL браузера:
   ```
   https://docs.google.com/spreadsheets/d/ЭТОТ_КУС_И_ЕСТЬ_ID/edit
   ```

---

## Шаг 2 — Создать Service Account (сервисный аккаунт)

1. Открыть [console.cloud.google.com](https://console.cloud.google.com)
2. Создать новый проект (или выбрать существующий)
3. В поиске найти **"Google Sheets API"** → нажать **"Включить"**
4. Перейти в **IAM и администрирование → Сервисные аккаунты**
5. Нажать **"Создать сервисный аккаунт"**
   - Имя: `gorshkov-sheets`
   - Нажать **"Готово"**
6. Кликнуть на созданный аккаунт → вкладка **Ключи**
7. **Добавить ключ → JSON** → скачается файл
8. Переименовать скачанный файл в **`credentials.json`**
9. Положить `credentials.json` рядом с `app.py`

---

## Шаг 3 — Дать доступ к таблице

1. Открыть скачанный `credentials.json`
2. Найти поле `"client_email"` — там будет адрес типа:
   ```
   gorshkov-sheets@ваш-проект.iam.gserviceaccount.com
   ```
3. Открыть Google Таблицу
4. Нажать **"Поделиться"** (кнопка вверху справа)
5. Вставить этот email → роль **Редактор** → нажать **"Отправить"**

---

## Шаг 4 — Прописать ID таблицы в app.py

Открыть `app.py`, найти строку:
```python
SPREADSHEET_ID = 'ВСТАВЬТЕ_ID_ТАБЛИЦЫ_СЮДА'
```
Заменить на ID из шага 1, например:
```python
SPREADSHEET_ID = '1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms'
```

---

## Шаг 5 — Установить зависимости и запустить

```bash
pip install -r requirements.txt
python app.py
```

Сайт будет доступен по адресу: **http://localhost:5000**
